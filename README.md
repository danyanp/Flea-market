# Flea_market青协跳蚤市场 （bomb版）

> 青协跳蚤市场是每一年一次的青协的活动
>> 学长，学姐将用不着的东西定价交给青协，青协将在跳蚤市场上卖掉，多出的差价用于青协公益活动
